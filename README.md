# twitter-trend-analysis-lda-nlp
## 📌 Project Overview

This project analyzes tweets shared by Donald Trump using Natural Language Processing (NLP), topic modeling (LDA), sentiment analysis, and time-series techniques.

The goal of this study is to uncover:

- Topic distributions within tweets
- Sentiment trends over time
- Temporal tweeting patterns
- Word co-occurrence relationships

---

## 📊 Dataset

The dataset was obtained from Kaggle and contains:

- Tweet text
- Date and time
- Media type
- Hashtags
- Retweet count
- Favorite count

---

## 🧠 Methods and Techniques

### 1️⃣ Text Preprocessing
- Lowercasing
- URL, mention and hashtag removal
- Stopword removal
- Punctuation cleaning
- Stemming

### 2️⃣ Exploratory Data Analysis
- Word Cloud visualization
- Hourly / Daily / Weekly tweet distribution
- Engagement analysis

### 3️⃣ Topic Modeling
- LDA (Latent Dirichlet Allocation)
- Topic interpretation and distribution analysis

### 4️⃣ Sentiment Analysis
- TextBlob polarity & subjectivity analysis
- Sentiment trend over time

### 5️⃣ Network Analysis
- Word co-occurrence network using NetworkX

---

## 📈 Key Findings

- Tweets are generally subjective and slightly positive in tone.
- Major topics revolve around:
  - Campaign slogans
  - America-related discourse
  - Republican rivals
  - Hillary Clinton
- Topic and sentiment trends fluctuate significantly during election periods.
